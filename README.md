GreekLetters
===========

Converts Greek alphabet and Julia LaTeX mappings to unicode characters in Sublime Text 2/3. It can also convert a list of mathematical key words to their unicode formatted character.

Julia unicode mapping based on https://docs.julialang.org/en/v1/manual/unicode-input/

Hitting `shift`-`tab` will replace the previous word with a utf-8 Greek letter, according to the following dictionary:
